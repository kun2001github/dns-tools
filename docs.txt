win调试用
python3.11
pip install flask dnspython
python app.py

部署使用(linux，以为win是使用不了gunicorn)
pip install gunicorn
启动命令
gunicorn -w 4 -b 0.0.0.0:8000 app:app
解释
-w 4：指定工作进程的数量，这里设置为 4 个，可以根据服务器的 CPU 核心数进行调整。
-b 0.0.0.0:8000：指定绑定的地址和端口，这里绑定到所有可用的网络接口（0.0.0.0），并监听 8000 端口。
app:app：指定 Flask 应用文件和应用实例的名称，前面的 app 是文件名 app.py，后面的 app 是 Flask 应用实例名。

使用docker（docker的name必须以小写）
docker build -t dns-tool .
docker run -p 8000:8000 dns-tool
